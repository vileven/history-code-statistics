# history-code-statistics
A set of tools to accumulate history statistics of code and show charts


example of usage 
```bash
npm i
SCAN_DIR="path/to/folder" GIT_BRANCH="master" PERIOD="31d" FROM_DATE="2021-07-01" LANGUAGES="Java,Kotlin" npm start
```
